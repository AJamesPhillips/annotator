The Annotator

The Annotator is a JavaScript tool that can be added to any webpage to allow annotation. Annotations can have comments, tags, users and more. See a live demonstration or more in the developers section below.

The Bookmarklet

The Bookmarklet is a simple tool that allows you to add the annotator to any webpage and save your annotations to AnnotateIt. Check out the Bookmarklet demo or get your very own bookmarklet right now by signing up for an account.

Developers

Adding the Annotator to a webpage is easy. First you'll need to download the source code and include it on your page along with jQuery. Then simply add the following line to load the annotator.

jQuery('#content').annotator();

To save the Annotations to the Store you'll need to sign up for an AnnotateIt account to get an Account Key and User Account. Then include the Store and Auth plugins.

Then sign in to AnnotateIt with the following code

jQuery('#content').annotator()
      .annotator('addPlugin', 'Store', {
        prefix: 'http://annotateit.org/api'
      })
      .annotator('addPlugin', 'Auth', {
          userId:      'myusername',        // Your AnnotateIt username
          consumerKey: 'some-big-string',   // Your Account Key
          authToken:   'some-bigger-string' // Your Auth Token
      });
